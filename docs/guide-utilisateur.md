# Guide Utilisateur

## Présentation

DetectIA est une application pédagogique autour de l'IA et de la désinformation. L'utilisateur peut :

- créer un compte élève ou professeur
- suivre des modules pédagogiques
- faire un quiz
- jouer à un mini-jeu
- gagner de l'XP
- débloquer des badges
- utiliser un chat IA

## Connexion et création de compte

### Compte professeur

Pour créer un compte professeur :
- aller sur la page d'inscription
- choisir `Compte professeur`
- utiliser obligatoirement une adresse `@prof.com`
- renseigner un mot de passe

Les autres domaines comme Gmail, Outlook ou Yahoo sont refusés pour ce type de compte.

### Compte élève

Pour créer un compte élève :
- aller sur la page d'inscription
- choisir `Compte eleve`
- saisir le code élève donné par le professeur
- compléter les autres champs

Le code élève n'est utilisable qu'une seule fois.

## Parcours professeur

Une fois connecté, le professeur accède au tableau de bord professeur.

Depuis ce tableau de bord, il peut :
- générer des codes d'invitation élève
- voir les codes générés
- consulter les réponses enregistrées par les élèves
- voir certains échanges ou erreurs liés au chat IA

Le tableau de bord professeur est masqué pour un compte élève.

## Parcours élève

Une fois connecté, l'élève accède principalement à :
- `Modules`
- `Chat IA`
- `Quiz`
- `Mini-jeux`
- `Badges`

Un élève ne voit pas le panel professeur.

## Modules pédagogiques

Dans la page `Modules` :
- chaque module peut être ouvert depuis la liste
- cliquer sur `J'ai lu tout le cours` permet de valider le module
- un module validé est marqué `Termine`
- l'XP liée au module est gagnée une seule fois

Des ressources externes peuvent aussi être ouvertes, mais elles sont facultatives.

## Quiz

Dans la page `Quiz` :
- les questions sont affichées une par une
- une bonne réponse est affichée en vert
- une mauvaise réponse est affichée en rouge
- à la fin, le score est affiché
- un quiz terminé peut ajouter de l'XP

## Mini-jeux

Dans la page `Mini-jeux` :
- le jeu principal est `Detective IA`
- l'utilisateur doit distinguer un contenu IA d'un contenu réel
- le score final donne de l'XP

## Badges

La page `Badges` affiche :
- les badges déjà débloqués
- les badges encore verrouillés
- les conditions de déblocage
- la date d'obtention des badges gagnés

## Chat IA

Le chat IA permet :
- de poser des questions sur l'IA
- d'envoyer jusqu'à 3 fichiers texte/code
- d'obtenir une réponse générée par IA

Rappels importants :
- ne pas envoyer de données sensibles
- toujours vérifier les informations importantes

## Navigation mobile

Sur mobile :
- la navigation principale est accessible en bas de l'écran
- un panneau complémentaire permet d'accéder aux autres liens
- le menu se ferme automatiquement après clic sur un lien
